# User Guide

## First run
1. Choose language.
2. Import legacy data or start fresh.
3. Configure projector and return monitors.
4. Finish onboarding and open dashboard.

## Core flows
- Use sidebar to navigate hymnal, Bible, presentations, services, and utilities.
- Use command palette (`Cmd/Ctrl + K`) for quick actions.
- Use shortcuts panel (`Cmd/Ctrl + /`) for keyboard workflows.

## Projection controls
- Open/close projector and return windows from status bar or command palette.
- Use black/logo screen toggles for transitions.
- Utility projections (timer/clock/lottery) are synchronized in projector/return/streaming.

## Streaming
- Open `Settings` and configure streaming port.
- Start server and share generated URLs.
- Keep broadcast enabled during services.

## Help
- Open `Help` route for guided tour and shortcut references.
