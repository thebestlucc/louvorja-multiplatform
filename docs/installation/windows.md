# Installing LouvorJA on Windows

## Download

1. Go to the [Releases page](https://github.com/nickksoares/louvorja-multiplataform/releases) and download the file ending in `-setup.exe` (e.g., `LouvorJA_0.1.0_x64-setup.exe`).

## Install

2. Double-click the downloaded `.exe` file to start the installer.

3. **If you see a "Windows protected your PC" warning (SmartScreen):**
   - This is normal for new software that hasn't been signed yet. Your download is safe.
   - Click **"More info"**
   - Then click **"Run anyway"**

4. Choose your installation mode:
   - **"Install for me only"** (recommended — no administrator password needed)
   - **"Install for all users"** (requires administrator password)

5. Follow the installer wizard and click **"Install"**.

6. When finished, click **"Launch"** or find **LouvorJA** in the Start menu.

## First launch

7. Select your preferred language (Portuguese, English, or Spanish).

8. Choose **"Start fresh"** or **"Import from Delphi version"** if you have an existing database.

## Verify your installation

9. Open the app and check the version number in the bottom status bar (e.g., `v0.1.0`).

## Automatic updates

LouvorJA checks for updates automatically. When a new version is available, a notification will appear at the bottom-right corner. During a worship service, the notification is held until the service ends — so your projection is never interrupted.

## Troubleshooting

- **Installer won't run:** Right-click the file → Properties → check "Unblock" → Apply, then try again.
- **App won't start:** Make sure you have Windows 10 or later. Restart your computer and try again.
- **Need help?** Open an issue at [github.com/nickksoares/louvorja-multiplataform/issues](https://github.com/nickksoares/louvorja-multiplataform/issues).
