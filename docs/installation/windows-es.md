# Instalacion de LouvorJA en Windows

## Descarga

1. Ve a la [pagina de releases](https://github.com/nickksoares/louvorja-multiplataform/releases) y descarga el archivo que termina en `-setup.exe` (ej: `LouvorJA_0.1.0_x64-setup.exe`).

## Instalar

2. Haz doble clic en el archivo `.exe` descargado para iniciar el instalador.

3. **Si aparece el aviso "Windows protegio su PC" (SmartScreen):**
   - Esto es normal para software nuevo que aun no ha sido firmado digitalmente. La descarga es segura.
   - Haz clic en **"Mas informacion"**
   - Luego haz clic en **"Ejecutar de todas formas"**

4. Elige el modo de instalacion:
   - **"Instalar solo para mi"** (recomendado — no necesita contrasena de administrador)
   - **"Instalar para todos los usuarios"** (requiere contrasena de administrador)

5. Sigue el asistente de instalacion y haz clic en **"Instalar"**.

6. Cuando termine, haz clic en **"Iniciar"** o busca **LouvorJA** en el menu Inicio.

## Primer uso

7. Selecciona tu idioma preferido (Portugues, Ingles o Espanol).

8. Elige **"Comenzar de cero"** o **"Importar de la version Delphi"** si tienes una base de datos existente.

## Verifica tu instalacion

9. Abre la aplicacion y verifica el numero de version en la barra de estado inferior (ej: `v0.1.0`).

## Actualizaciones automaticas

LouvorJA busca actualizaciones automaticamente. Cuando una nueva version este disponible, aparecera una notificacion en la esquina inferior derecha. Durante un servicio de adoracion, la notificacion se pospone hasta que termine el servicio — tu proyeccion nunca sera interrumpida.

## Solucion de problemas

- **El instalador no se ejecuta:** Haz clic derecho en el archivo → Propiedades → marca "Desbloquear" → Aplicar, y vuelve a intentar.
- **La app no abre:** Asegurate de tener Windows 10 o superior. Reinicia el computador e intenta de nuevo.
- **Necesitas ayuda?** Abre un issue en [github.com/nickksoares/louvorja-multiplataform/issues](https://github.com/nickksoares/louvorja-multiplataform/issues).
