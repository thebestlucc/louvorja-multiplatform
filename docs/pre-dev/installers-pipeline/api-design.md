## Contract 2: Error Classifier (Internal)

**Interface:** Pure function, frontend-only.

```typescript
type UpdateErrorCategory = "network" | "disk_space" | "permission" | "generic";

interface PastoralError {
  titleKey: string;    // e.g., "updater.errorNetwork"
  whyKey: string;      // e.g., "updater.errorNetworkWhy"
  actionKey: string;   // e.g., "updater.errorNetworkAction"
  reassuranceKey: string; // always "updater.errorDataSafe"
}

function classifyUpdateError(error: unknown): PastoralError;
```

**Mapping:**

| Category | Pattern Match | Title Key | Why Key | Action Key |
|----------|--------------|-----------|---------|------------|
| network | "network", "connection", "timeout", "DNS" | `updater.errorNetwork` | `updater.errorNetworkWhy` | `updater.errorNetworkAction` |
| disk_space | "space", "disk", "ENOSPC" | `updater.errorDiskSpace` | `updater.errorDiskSpaceWhy` | `updater.errorDiskSpaceAction` |
| permission | "permission", "access", "EACCES" | `updater.errorPermission` | `updater.errorPermissionWhy` | `updater.errorPermissionAction` |
| generic | (fallback) | `updater.errorGeneric` | `updater.errorGenericWhy` | `updater.errorGenericAction` |

All categories share: `reassuranceKey = "updater.errorDataSafe"`
